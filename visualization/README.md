# Hero animation — technical note

The banner at the top of the profile README is generated by
[`simulate_metacommunity.py`](simulate_metacommunity.py). It is a **conceptual
simulation**: it illustrates a class of ecological dynamics I work on. It is not
an empirical result and the fields are not fitted to data.

## What is being shown

A spatially explicit, trait-structured metacommunity on a periodic lattice
(450 × 166 cells, a torus, so there are no boundary artefacts). Two state
variables are tracked in every cell:

| Channel | Variable | Meaning |
| --- | --- | --- |
| Colour (viridis) | `z(x, y, t)` | community-mean trait value |
| Brightness | `B(x, y, t)` | biomass / occupancy density |

Two static, spatially autocorrelated layers set the template:

- `E(x, y)` — the local trait optimum, which drives **environmental filtering**
- `K(x, y)` — the local carrying capacity, which creates **refugia**

## Model

Biomass follows a stochastic reaction–diffusion equation with a Holling type III
loss term — the canonical form used for ecosystems with alternative stable
states (Noy-Meir 1975; May 1977; Scheffer et al. 2001):

```
dB/dt = r B (1 − B/K) M(z, E)  −  S(t) B² / (B² + h²)  −  m B
        + D ∇²B  +  σ_B √B ξ(x, y, t)
```

The community-mean trait is carried as the first moment `P = B z`, so that
offspring inherit the parental trait, dispersal transports trait *flux* rather
than trait value, and recolonised cells inherit the trait composition of their
immigrants:

```
d(Bz)/dt = z · [growth and loss terms]        offspring inherit z
         + B · V (E − z) / w²                 directional selection toward E
         + D ∇²(Bz)                           trait flux under dispersal
         + σ_z √B ξ                           mutation / drift
```

with the environmental filter

```
M(z, E) = exp( −(z − E)² / (2 w²) )
```

Seven discrete disturbance events per loop clear biomass in soft-edged patches.

## The regime shift

`S(t)` is a slow, strictly periodic environmental stress driver (shown as the
sparkline in the legend strip). Because the type III loss term makes the local
dynamics bistable, `S(t)` carries the landscape across a **fold bifurcation**:

1. **Patterned state** — near-continuous cover; colour tracks `E`, but dispersal
   prevents perfect local adaptation (a migration–selection balance), so `M < 1`
   along steep environmental gradients.
2. **Collapse** — as `S` rises past the tipping point, cover fragments rapidly.
   Loss is *patchy*, not uniform: low-`K` and poorly-adapted cells go first.
3. **Degraded state** — biomass persists only in high-`K` refugia. Landscape
   cover falls from ~99% to ~23%.
4. **Hysteretic recovery** — recolonisation begins only once `S` has fallen well
   *below* the collapse threshold. Dispersal fronts spread out of the refugia
   carrying refugium trait values, so recolonised ground is transiently
   maladapted before selection re-tunes it to the local optimum.

The hysteresis is real and measurable in the run: at the same stress level
`S ≈ 0.21`, cover is ~0.89 on the way up and ~0.30 on the way down.

## Seamless looping

Stochastic simulations do not naturally return to their initial state, so the
loop is closed by construction rather than by cross-fading:

- Every stochastic input — the noise field and the disturbance schedule — is
  pre-generated for **exactly one loop period** and replayed identically on each
  pass. The noise is smoothed circularly in *x*, *y* **and** *t*, so it wraps.
- The system is therefore a deterministic dynamical system under strictly
  periodic forcing. Being dissipative, it converges onto a periodic orbit.
- Four burn-in loops are integrated and only the fifth is recorded.

The residual state mismatch across the seam is printed at the end of the run.
For the committed asset it is **0.0000%** (relative mean |ΔB|), i.e. the last
frame flows into the first exactly like any other frame transition.

## Design choices

- **viridis**, trimmed to `[0.22, 1.0]` — perceptually uniform and monotonic in
  luminance, so it survives deuteranopia, protanopia and greyscale printing.
- **Dark neutral background** (`#0b0e16`), self-contained, so the panel reads
  the same in GitHub light and dark themes.
- **No quantitative axes.** The only annotated quantity is `S(t)`, which is
  genuinely the model's driver, and it is drawn without numeric tick labels.
- **Animated WebP**, not GIF. These are continuous, noisy fields: GIF's 256-colour
  LZW encoding produces 9–16 MB with visible banding, while animated WebP gives
  full 24-bit colour at 2.2 MB.

## Reproducing

```bash
pip install -r requirements.txt
python simulate_metacommunity.py
```

Runtime is roughly 60 s on a laptop CPU. Output is written to `../assets/`.
The run is fully deterministic given `SEED = 20260419`.

Useful flags: `--frames`, `--fps`, `--scale`, `--quality`, `--burn`,
`--still-frame`, `--also-gif`.
